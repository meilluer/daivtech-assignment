A movie booking sample app 

steps to run -

1.download zip of repo

2.unzip repo 

3.open folder in command prompt 

4.run "npm install expo"

5. run "npx expo start"
   
6.on android / ios devices download expo go and scan the QR code 

NOTE- you must run "npx expo start" on root of project ( make sure package.json is present )

Alternatively for Android users a direct APK is also present in "releases" section , download and install apk
